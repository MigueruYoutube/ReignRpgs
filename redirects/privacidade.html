Política de Privacidade - Reign of RPGs

Última atualização: 30 de Junho de 2025

Este site é um projeto do servidor Reign of RPGs, criado por Miguel Kayky Nascimento Costa.

Coleta de Dados

Nosso site foi desenvolvido com o máximo respeito à sua privacidade:

  • Não solicitamos login, cadastro ou informações pessoais
  • Nenhuma informação sensível do usuário é coletada ou armazenada
  • Qualquer dado de navegação ou armazenamento é controlado única e exclusivamente pelo navegador utilizado pelo usuário

Hospedagem e Transparência

O site é hospedado via GitHub Pages, o que garante:

  • Não existe qualquer envolvimento de terceiros além do próprio GitHub
  • Todo o código-fonte e arquivos públicos estão disponíveis diretamente no GitHub oficial
  • Transparência total: qualquer informação carregada pelo site pode ser verificada publicamente

Rastreamento e Cookies

Este site não possui:

  • Ferramentas de rastreamento externo
  • Cookies personalizados
  • Sistemas de coleta ou análise de dados

Aceitação dos Termos

Ao utilizar este site, você concorda com estes termos básicos de privacidade e compreende que nenhum dado pessoal está sendo coletado ou armazenado.
